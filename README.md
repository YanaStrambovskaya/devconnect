# 🚀 DevConnect

**DevConnect** is a developer-focused social platform where engineers can showcase projects, build professional profiles, and connect with other developers.

---

## ✨ Features

- 👤 User authentication (registration & login)
- 🧑‍💻 Developer profile creation and editing
- 📂 Project showcase system
- 📰 Activity feed (posts / updates)
- 🔍 Search developers and skills (Comming soon)
- ⚡ Responsive UI for all devices (Comming Soon)

---

## 🛠 Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend / BaaS
- Firebase (Auth, Database, Storage)

### Tools
- Git & GitHub
- REST / Firebase API
