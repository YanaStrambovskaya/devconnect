# 🚀 DevConnect

**DevConnect** is a developer-focused social platform where engineers can showcase their projects, build professional profiles, and connect with other developers.

---

## ✨ Features

* 👤 User authentication & profiles
* 🧑‍💻 Create and edit developer profiles
* 📂 Showcase projects
* 📰 Feed with posts / updates
* 🔍 Search for developers and skills
* ⚡ Responsive UI

---

## 🛠 Tech Stack

**Frontend**

* React
* TypeScript
* Tailwind CSS

**Backend**

* *Firebase*

**Other**

* REST API / Firebase
* Git & GitHub
